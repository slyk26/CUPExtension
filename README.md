# IPP / CUPS printing for Chrome & Chromebooks

[my favorite extension](https://chromewebstore.google.com/detail/ipp-cups-printing-for-chr/lkhfeoafdgbaecajkdbioenncjopbpmk)
is no longer supported so I asked my ol' AI friend to update to manifest v3

## how to install
- download zip or crx from release
- unzip / extract file
- go to ``brave://extensions/`` or ``chrome://extensions/`` and enable developer mode top right
- select `Load unpacked` on left side and select unzipped directory
